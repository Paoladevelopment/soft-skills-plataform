Generate an easy "paraphrase" listening challenge.

ACTIVE PROMPT TYPE: {prompt_type}
ACTIVE AUDIO LENGTH: {audio_length}

Use ONLY the rules in the section whose header exactly matches the ACTIVE PROMPT TYPE.
Ignore all other sections.

OBJECTIVE
Create TWO short texts:  
1. **audio_text** → the original passage that will be spoken aloud.  
2. **reference_text** → a paraphrased version that keeps the same meaning but changes wording or structure slightly.  

Both texts must be appropriate for {audio_length} and written entirely in the language specified by {locale} (e.g., “es” = español).  
The **audio_text** should sound natural and fluent when read aloud, using simple and clear language suitable for educational listening.  

──────────────────────────────
SECTIONS BY PROMPT TYPE (pick exactly one based on ACTIVE PROMPT TYPE)

[descriptive]
- Write an original description of an object, place, or moment in the **audio_text**.
- Then write a **reference_text** that expresses the same visual or contextual meaning using different but simple words.
- Avoid poetic or emotional phrasing; focus on clear imagery.

[historical_event]
- Write an original **audio_text** describing a real historical event or process (without using dates or names).
- Then create a **reference_text** that conveys the same event with basic rewording.
- Maintain historical accuracy, sequence, and neutrality.

[instructional]
- Write an **audio_text** that explains a simple process, procedure, or step-by-step action (e.g., how to plant a seed, cook rice, or clean a room).
- Use imperative or procedural tone (verbs like “prepare,” “mix,” “open,” “check”).
- Then write a **reference_text** that gives the same instructions with slightly different phrasing but identical meaning.
- Ensure both versions maintain logical order, clarity, and educational tone.

[dialogue]
- Write a short, natural two-speaker exchange labeled exactly as:
  "Speaker 1": <text>
  "Speaker 2": <text>
- Alternate turns clearly with line breaks ("\n").
- The **audio_text** must contain at least one full exchange (2–3 turns total).
- Then write a **reference_text** that paraphrases the same conversation with different wording but keeps the same intent and natural flow.
- Both dialogues must sound like real conversation — friendly, spontaneous, and coherent.
- Each speaker label ("Speaker 1", "Speaker 2") must appear exactly as shown.
- Do NOT omit the speaker labels in either audio_text or reference_text.


[narrated_dialogue]
- Write a short narration summarizing a conversation (third-person voice, no "Speaker 1" labels).
- Then create a **reference_text** that rephrases the narration while preserving meaning and context.
- Maintain clear structure and natural spoken tone.

──────────────────────────────
OUTPUT REQUIREMENTS
You must return a structured response with these exact fields:
- audio_text: Original passage that sounds natural when spoken aloud
- reference_text: Paraphrased version with the same meaning
- rubric: Array with evaluation criteria for assessing paraphrase quality

──────────────────────────────
RULES AND VALIDATION
- audio_text:
  - Must be the original passage.
  - Must sound fluent and natural for TTS.
  - Must align with the ACTIVE PROMPT TYPE’s structure.
  - Must align with {audio_length} defined in the base system.

- reference_text:
  - Must have the same meaning as audio_text.
  - Must change some words or phrasing but preserve logical and grammatical clarity.
  - Must not shorten or expand excessively (length within ±15% of original).

- rubric:
  - Must ALWAYS be exactly this array (do not modify wording or order):
    ["semantic equivalence", "fluency and naturalness", "clarity and simplicity"]

Avoid:
- Introducing new facts or altering meaning.
- Using uncommon or technical vocabulary.
- Changing tone (e.g., from instructional to narrative).
- Metadata, comments, or Markdown formatting.

──────────────────────────────
EXAMPLE (for reference only):
- audio_text: "Para hervir el agua, coloca una olla sobre la estufa y enciende el fuego medio."
- reference_text: "Pon una cacerola en la estufa y caliéntala a fuego medio para que el agua hierva."
- rubric: ["equivalencia semántica", "fluidez y naturalidad", "claridad y simplicidad"]

- reference_text: "\"Speaker 1\": ¡Qué gusto verte! ¿Cómo te va?\n\"Speaker 2\": Todo bien. Recordé que mencionamos ir a la playa la vez pasada.\n\"Speaker 1\": Me parece una buena idea hacerlo este fin de semana, podríamos llevar algo para comer."
